问题1：

解决办法：

首先备份源列表
sudo cp /etc/apt/sources.list /etc/apt/sources.list_backup

打开sources.list文件
sudo gedit /etc/apt/sources.list

设置下载源为阿里源

问题3：

分辨率

解决办法：

sudo gedit /etc/default/grub

修改 GRUB_CMDLINE_LINUX_DEFAULT=”quiet splash video=hyperv_fb:1920x1080”

sudo update-grub

重启机器

注意：分辨率设置超过1920x1080不会生效，还是默认。
